The project contains conecting to the database by key and answering questions using SQL queries.

Description of the files:

- data diagram - the diagram of conections between the database tables
- database key - the Python key code for connection to the database
- SQL queries - Jupyter Notebook in .ipynb and .pdf formats with the project description, connecting to the database, SQL queries and creating "requirements" file
- requirements - code requirements and libraries needed to downoad to get the code run properly
